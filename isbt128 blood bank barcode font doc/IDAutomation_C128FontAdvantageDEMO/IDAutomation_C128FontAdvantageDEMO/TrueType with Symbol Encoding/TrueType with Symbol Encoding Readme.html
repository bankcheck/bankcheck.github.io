TrueType with Symbol Encoding Fonts

Symbol encoded fonts are provided in this package for the primary purpose of being able to properly view the font name in font drop-down selection lists that exist in popular applications such as Microsoft Office. Symbol fonts have been known to cause problems when the locale of the operating system is not set to US Western (ANSI 1252). They have also been known to cause problems with Crystal Reports and when embedding into PDF documents. Symbol fonts are usually only available in TTF form and do not include sizes larger than “L”. If problems are experienced, including the appearance of boxes within the barcode, do not use symbol encoded fonts.
