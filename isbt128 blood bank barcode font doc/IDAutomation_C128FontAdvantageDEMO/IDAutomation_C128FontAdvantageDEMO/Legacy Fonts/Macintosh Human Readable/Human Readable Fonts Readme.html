Human Readable (HR) Fonts

Only one size is available in the DEMO version of the human readable fonts.
To order the complete Code 128 Font Package: https://www.idautomation.com/barcode-fonts/code-128/

The human readable fonts in this folder are not normally used, because the Code 128 check digit is usually visible in the result. These versions of HR fonts are often used for testing purposes or when it is not a problem to have the symbology check digit displayed.

To print the human readable data without the check digit, use the fonts beginning in IDAutomationC128 and print the data encoded below the barcode as a separate field. The XXS size is not available in human readable versions.
