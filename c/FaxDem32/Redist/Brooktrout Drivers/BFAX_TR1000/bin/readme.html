Welcome to the Bfv API SDK System Software
for TR1000, TR1034, and SR140 Products

The Brooktrout computer telephony products provide a wide range of reliable, high-performance PCI models for both voice and fax applications in the enterprise market and for the service provider.

The Brooktrout fax products consist of the following:

  • TR1000: High-performance voice, speech and fax resource platform
  • TR1034: Advanced 33.6 kilobit per second intelligent fax boards for TDM and T.38 for IP networks
  • SR140: High-performance host-based T.38 fax for IP networks

This software also supports economical TruFax-branded versions of the Brooktrout BRI and Analog fax products.

Please see the documentation accompanying your communications software for more information on the use of Brooktrout system software.
