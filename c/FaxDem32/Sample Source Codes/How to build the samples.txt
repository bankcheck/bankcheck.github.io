The source code of the sample applications is located in the “Sample Source Codes” folder. 
The  “Sample Source Codes” folder is installed depending on the product, by default to:
“C:\Program Files\BlackIce\FaxDem32\Sample Source Codes”  




The contents of the “Sample Source Codes”:

“DLL” Folder    – C++ samples, using the DLLs
“DOTNET” Folder - .NET samples 
“OCX” Folder    – Samples using the FaX.OCX and Voice.OCX.
